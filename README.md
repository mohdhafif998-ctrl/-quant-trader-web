# Quant Trader Web

Web app Streamlit untuk backtest strategi crypto.

## Cara guna di Streamlit Cloud

1. Upload semua fail ini ke GitHub repository baru.
2. Buka https://streamlit.io/cloud
3. Login guna GitHub.
4. Pilih repository.
5. Main file path: `app.py`
6. Tekan Deploy.

## Fail utama

- `app.py` = interface website
- `backtest.py` = engine backtest
- `indicators.py` = kira MA, RSI, ATR
- `optimizer.py` = cari parameter terbaik
- `requirements.txt` = library Python

## Nota

Ini untuk backtest sahaja. Jangan sambung API trading sebenar dulu.
